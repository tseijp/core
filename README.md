## Get Started

- `git clone https://github.com/tseijp/core`
- `cd core`, `npm i` and `npm start`
- open browser and visit [http://localhost:3000](http://localhost:3000)
- Now you can go to our [demo](https://tsei.jp/hook/), and try its usage.
